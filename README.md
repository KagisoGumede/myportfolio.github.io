## Setup Instructions

### 1. Basic Setup (No Server Required)

Simply double-click on `index.html` to open it in your browser!

### 2. Add Your Profile Image

Replace the placeholder image by:
- Saving your photo as `profile-image.jpg` in the same folder as index.html
- Or update the `src` attribute in index.html (line 38) with your image path

### 3. Contact Form Setup

The contact form uses two methods:

**Method 1: FormSubmit.co (Recommended)**
- The form is already configured to send to Mzawu739@gmail.com
- On first submission, FormSubmit will send a confirmation email
- Click the confirmation link to activate the form
- No registration or API keys needed!

**Method 2: Mailto Fallback**
- If FormSubmit fails, it automatically opens your default email client
- Works offline but requires email client to be configured

### 4. Customization

**Update Your Information:**
- Open `index.html` in any text editor
- Update text content in sections:
  - Hero section (lines 34-38)
  - About Me (lines 50-72)
  - Experience (lines 143-155)
  - Education (lines 159-167)
  - Skills (lines 100-140)
  - Projects (lines 230-260)

**Change Colors:**
- Open `styles.css`
- Modify CSS variables at the top (lines 1-12):
  ```css
  --primary-color: #0a4d5c;
  --secondary-color: #147a8c;
  --accent-color: #1ba3c6;
  ```

**Add More Projects:**
- Copy a project card in index.html
- Paste and update the content
- The carousel will automatically adjust

## Hosting Your Portfolio

### Option 1: GitHub Pages (Free)
1. Create a GitHub account at github.com
2. Create a new repository named `your-username.github.io`
3. Upload all files
4. Your site will be live at `https://your-username.github.io`

### Option 2: Netlify (Free)
1. Go to netlify.com
2. Drag and drop your folder
3. Get a free subdomain or connect your own domain

### Option 3: Vercel (Free)
1. Go to vercel.com
2. Import your GitHub repository
3. Automatic deployments on every update

## Troubleshooting

**Q: My profile image doesn't show**
A: Make sure the image file is in the same folder as index.html and named exactly `profile-image.jpg` (case-sensitive)

**Q: Contact form doesn't work**
A: First submission requires email confirmation from FormSubmit. Check your inbox at Mzawu739@gmail.com

**Q: Site looks broken on mobile**
A: Make sure you're viewing the actual files, not just the code. Open index.html in a browser.

**Q: How do I edit the content?**
A: Open index.html in any text editor (Notepad, VS Code, etc.) and modify the text between the HTML tags
